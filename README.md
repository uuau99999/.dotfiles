## Hoyup .dotfiles

#### Usage

     1. install stow
     2. cd ~/ && git clone https://github.com/uuau99999/.dotfiles
     3. stow .

#### tmux conf setup

     1. cd ~/.dotfiles/tmux
     2. ./prepare.sh
     3. enter tmux and enter `prefix + I` to install plugins, then run `prefix + r` to reload
